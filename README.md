# String Reversal Task

This repository contains a JavaScript function to reverse a given string.

## Problem Statement

Write a JavaScript function that takes a string as input and returns the string reversed.

**Example**:  
Input: `"hello"`  
Output: `"olleh"`

## Solution

The solution involves using the following steps:
1. Split the string into an array of characters using `split('')`.
2. Reverse the array using `reverse()`.
3. Join the reversed array back into a string using `join('')`.
